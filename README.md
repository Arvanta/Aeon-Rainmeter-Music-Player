Aeon – Minimalist Music Player

Aeon is a sleek, modern, and highly customizable music player skin for Rainmeter, designed with a focus on simplicity and aesthetic elegance. It features a clean interface that seamlessly integrates with your desktop, offering multiple style variations to match any wallpaper.


Requirements: To function correctly, this skin requires the WebNowPlaying plugin by keifufu to bridge your media playback with Rainmeter.
