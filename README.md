# Aeon – Minimalist Music Player

**Aeon** is a sleek, modern, and highly customizable music player skin for [Rainmeter](https://www.rainmeter.net/), designed with a focus on simplicity and aesthetic elegance. It features a clean interface that seamlessly integrates with your desktop, offering multiple style variations to match any wallpaper.

![alt text](https://github.com/Arvanta/Aeon-Rainmeter-Music-Player/blob/main/Aeon.png?raw=true)

## ✨ Features
- Sleek, modern, and minimalist design.
- Multiple style variations (Dark, Light, Transparent, and Windows 11 Mica) to match any background.
- Clean and intuitive user interface.
- Lightweight and highly customizable.

## ⚙️ Requirements
To function correctly, this skin requires the **[WebNowPlaying](https://github.com/keifufu/WebNowPlaying-Rainmeter)** plugin by *keifufu* to bridge your media playback with Rainmeter. 

## 🚀 Installation
1. Download and install the latest version of [Rainmeter](https://www.rainmeter.net/).
2. Install the required [WebNowPlaying plugin](https://github.com/keifufu/WebNowPlaying-Rainmeter).
3. Download the latest `.rmskin` package from the [Releases](../../releases) tab.
4. Double-click the downloaded `.rmskin` file to install the skin.
5. Open Rainmeter and load your preferred Aeon style.
